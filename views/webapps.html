KEXP Live Playlist


Role: Front and Back end developer
For one class project, we were tasked with making a full stack application using node and express. I elected to use KEXP's public song api to create a real-time playlist that allows users to listen and add songs to their profile that they like. Users can also list a particular song as their "Super favorite". This app was built with a mobile first design.

Pub Crawl


Role: Back end developer and git master
For a group project we buit a social networking site around pub and bar crawls. Users can create an event with a start time and date, and add locations to create their own pub crawl. Other usesr can join this event. Admins mark locations off as the move from pub to pub, keeping users informed of where everyone is at any given time. The app also queries Yelp for local bars, and uses the Google API to display a map of all the locations
